# prokopto.dev
Personal website repository, with automatic deployment to server via Digital Ocean App platform.

## Licenses
The content of this website is shared and available under an Creative Commons License Attribution-Share-Alike license; this includes posts, images of my creation or providence, papers, PDFs, etc.

Hugo, Go, and the hugo-coder theme each maintain their own license structure, and so the actual structure of the website falls under them. Any changes made to the structure of the site persist under those licenses.

## Content

Content is developed on a branch-by-branch basis, with each post (after the test post), being created (or updated) via a branch.

Commits are squashed upon merge with the main branch (for deployment).
